# Configuration Supabase Storage (logo / favicon)

Le bucket `assets` est créé automatiquement au démarrage du serveur.

Si l'upload échoue, créez-le manuellement :

1. [Supabase Dashboard](https://supabase.com/dashboard) > Votre projet
2. **Storage** > **New bucket**
3. Nom : `assets`
4. **Public bucket** : activé (pour que les images soient accessibles)
5. Créer
